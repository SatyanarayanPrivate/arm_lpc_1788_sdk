Readme.txt for BSP for IAR LPC1788-SK start project.
This project was built for IAR Workbench for ARM V6.21.

Supported hardware:
===================
The sample project for the NXP LPC1788 CPU
is prepared to run on a IAR LPC1788 eval board,
but may be used on other target hardware as well.

Configurations
==============
- Debug
  This configuration is prepared for download into
  internal Flash using J-Link.

- Release
  This configuration is prepared for download into
  internal Flash using J-Link.

Included middleware components (trial versions):
================================================
- emWin V5
  - (3.00.01) emWin BASIC color
  - (3.01.00) emWin WM/Widgets (window manager and widgets)
  - (3.01.02) emWin Memory devices (memory devices for flicker-free animation)
  - (3.01.03) emWin Antialiasing (antialiasing smoothes curves and diagonal lines)
  - Drivers:
  - (3.10.02) emWin LCDLin (supported color depths: 1, 2, 4, 8, 15, 16, 32 bpp)
